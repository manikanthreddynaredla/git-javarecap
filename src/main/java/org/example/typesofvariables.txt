Java has **three types of variables**:

1. **Local Variables**
2. **Instance Variables (Non-static fields)**
3. **Static Variables (Class variables)**

* Not accessible outside the method

## ğŸ”¹ **2. Instance Variables (Object / Non-static variables)**

Declared **inside a class** but **outside any method**.

### âœ” Key Points

* Belong to **each object**
* Every object gets a **separate copy**
* Default values are assigned (0, null, false)
* Stored in **heap memory**

### âœ” Example

```java
public class Student {
    String name;     // instance variable
    int marks;       // instance variable

    public static void main(String[] args) {
        Student s1 = new Student();
        s1.name = "John";
        s1.marks = 90;

        Student s2 = new Student();
        s2.name = "Alex";
        s2.marks = 85;

        System.out.println(s1.name + " " + s1.marks);
        System.out.println(s2.name + " " + s2.marks);
    }
}
```

### âœ” Use Cases

* Storing object-specific data
* Example: name/age of a person, balance of bank account

### âœ” Interview Tip

â€œInstance variables get memory only when an object is created.â€

## ğŸ”¹ **3. Static Variables (Class-Level variables)**

Declared with the **static keyword**.

### âœ” Key Points

* Shared by **all objects** of the class
* Memory allocated only **once** (class loading time)
* Access using **ClassName.variable**

### âœ” Example

```java
public class Employee {
    static String company = "TCS";  // static variable
    String name;                    // instance variable

    Employee(String name) {
        this.name = name;
    }

    void display() {
        System.out.println(name + " works at " + company);
    }

    public static void main(String[] args) {
        Employee e1 = new Employee("Ravi");
        Employee e2 = new Employee("Kiran");

        e1.display();
        e2.display();
    }
}
```

### âœ” Use Cases

* Common/shared information
* Constants:

  ```java
  static final double PI = 3.14159;
  ```
* Counters:

  ```java
  static int count;
  ```

### âœ” Interview Tip

â€œStatic variables are created in method area, shared across all objects.â€
